フォント設定

このダイアログでOmegaTがソース、翻訳、参考訳文、用語集を表示するために使うフォントを変更できます。これはメインメニューから設定 -> フォント設定で呼び出されます。

このダイアログには以下のものがあります:

  • マシンで利用可能なフォントから一つを選択するためのドロップダウンリスト。
  • フォントサイズを変更するためのスピンボックス。
  • 選択したフォントをその場で確認するために同じ文章を表示している部分。

警告!特に多くの分節を持つ大きなファイルをエディタで開いていると、フォント設定を変更した後表示が更新されるまでにかなり長い時間がかかることがあります。また、言語の組み合わせによってはよりよい振る舞いをするフォントもあります。一般的に言うと、プロジェクトを開く前にフォントを変更した方が良いです。このペナルティは G4 + Mac OS X + 日本語の組み合わせのときに最大になります。
