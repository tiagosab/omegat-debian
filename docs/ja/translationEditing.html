翻訳文書編集

翻訳は分節から分節に移動しながら行い、一度には一つの分節を編集します。分節間を移動するときに、編集フィールドが他の文章と置き換えられることがあります。例えば、既に翻訳した分節に移動すると、編集フィールドは常に既存の翻訳に置き換えられるので、以前の翻訳を編集できます。

しかし、まだ翻訳されていない分節に移動した場合、OmegaT は設定されたテキストを分節に設定できるので、好みの方法で編集できます:

  • まず、自動的に編集フィールドに原文を設定できます。これは、商標や固有名詞などのそのまま残しておく必要のあるものが多く含まれている文書では極めて便利です。設定 -> 翻訳入力行...に移動して原文ラジオボタンが選択されていることを確認して下さい。
  • または、編集フィールドを空白のままにしておくこともできます。このオプションにより、原文テキストを削除することなく翻訳を入力できるので、2 キーストローク (Ctrl+A と Del) を節約できます。設定 -> 翻訳入力行...に移動して何も入れませんラジオボタンを選択します。
  • あるいは、今から編集しようとしているものに最も似ている文字列の翻訳を挿入することもできます。設定 -> 翻訳入力行...に移動して、最も一致率の高い参考訳文のフラグを選択して、自動的に挿入するための最小一致率を調整してください。

OmegaT で以下のいずれかの設定を行なう場合は、設定 -> 翻訳入力行...に移動して...

  • 原文ラジオボタンが選択されていることを確認して下さい。これはデフォルトで、ほとんどの場合これが正解です。
  • 何も入れませんラジオボタンを選択して下さい。
  • 最も一致率の高い参考訳文のフラグを選択して、自動的に挿入するための最小一致率を調整してください。また、参考訳文を挿入したことを区別するための前置詞を変更することもできます。

... そしてOKをクリックして下さい。

保存する文書

翻訳する必要のあるドキュメントには、商標や固有名詞など、翻訳されたドキュメントでも同じである必要があるものも含まれています。このような変化しない文書を含む分節については二つの戦略があります。

まず、そのような分節を翻訳しないままにしておくことができますが、OmegaT はそのような翻訳していない分節を通知するので、後の翻訳レビューの時に、変化しないものであることを思い出せずに翻訳してしまうかもしれません。これはデフォルトです。

二つ目は、たとえ訳文が原文と同じでも訳文を保存するように OmegaT に指示することで、こうすると OmegaT は文書を見直して、同じ内容に翻訳することにしたことを思い出させてくれます。設定 -> 翻訳入力行...に移動して原文と等しい訳文を許可するフラグを選択します。
