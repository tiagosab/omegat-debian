Az OmegaT telepítése

Az OmegaT telepítéséhez mindössze csomagolja ki a letöltött állományt. Ezzel az adott könyvtárban létrejön az omegat/ könyvtár, amelyben megtalálja az OmegaT futtatásához szükséges összes állományt.

A .tar.gz állomány kicsomagolása:

$ tar xf letöltött_állomány.tar.gz

Az OmegaT menükhöz (KDE) vagy panelekhez (Gnome) való hozzáadása

KDE-felhasználók

Az OmegaT-t az alábbiak szerint adhatja hozzá menüihez:

  • Control Center - Desktop - Panels - Menus - Edit K Menu - File - New Item/New Submenu
  • Ezt követően egy megfelelő menü kiválasztása után adjon hozzá egy almenüt/menüpontot: File - New Submenu and File - New Item. Az új menüpont nevének adja meg az OmegaT-t.
  • A "Command" mezőben az iránybillentyűvel keresse ki az OmegaT futtatási kódját (a kicsomagolatlan könyvtárban az OmegaT nevű állományt) és válassza ki.
  • Kattintson az ikonra (a Name/Description/Comment mezőktől jobbra) - Other Icons - Browse, majd térjen át az OmegaT-alkalmazás mappájában az /images almappába. Válassza ki az OmegaT.png ikont.
  • Végül a File - Save segítségével mentse el a módosításokat.

Gnome-felhasználók

Az OmegaT-t hozzáadhatja paneljéhez (a képernyő tetején lévő sávhoz) az alábbiak szerint:

  • A jobb gombbal kattintson a panelre - Add New Launcher.
  • A "Name" mezőben adja meg az "OmegaT"-t; a "Command" mezőben az iránybillentyűvel keresse ki az OmegaT futtatási kódját (a kicsomagolatlan könyvtárban az OmegaT nevű állományt). válassza ki, majd kattintson az OK-ra.

Az OmegaT működtetése

Ha telepítette az OmegaT-t, elindíthatja az alkalmazást a parancssorból, vagy létrehoz egy kódrészletet a parancssori futtatási paraméterekkel, vagy közvetlenül az OmegaT.jar-ra kattintva elindíthatja az alkalmazást. A módszer az adott csomagtól függően változhat. Ellenőrizze, hogy a PATH beállításai helyesek-e és a .jar-fájlok megfelelően vannak-e társítva a Java futtató programjához.

Indítás a parancssorból

Általában az OmegaT-t nem szükséges a parancssorból elindítani. Ugyanakkor az OmegaT-t parancssorból való elindításával módosíthatja a program viselkedését. Az OmegaT-t kétféleképpen indíthatja el parancssorból.

A Mac OSX csomag felhasználóinak a parancssorból való indításhoz ajánlatos az Egyéb rendszerek alatt tárgyalt csomagot választani. Lásd fent.

1. módszer: parancssori ablak megnyitása

A parancssori ablak másik elnevezése a "terminálablak". Windows alatt "Parancssor" a neve és a Start-menü/Minden program/Kellékek menüpontból érhető el. Mac OSX alatt ennek az Applications → Utilities útvonalon elérhető Terminal a megfelelője.

Az OmegaT elindításához általában két parancsot kell beírnia. Az első:

cd {folder}

ahol a {folder} azon mappa neve teljes útvonallal, ahol az OmegaT program, konkrétan az OmegaT.jar található. Ennek megfelelően a parancs formája az alábbihoz lesz hasonló:

Windows esetén
cd C:\Program Files\OmegaT
MacOSX esetén
cd /Applications/OmegaT
Linux esetén:
cd /usr/local/omegat

Ez a parancs az aktuális mappát az OmegaT futtatható állományát tartalmazó mappára módosítja. A második parancs indítja el voltaképpen az OmegaT-t. Legelemibb formájában a parancs a következő:

java -jar OmegaT.jar

Ügyeljen a nagybetűkre, mert értékük nem azonos a kisbetűkével!

Ez a módszer előnyös, mivel alkalmas hibaszűrésre: ha a program használata során hiba lép fel, a terminálablakban megjelenik egy hibaüzenet, amely tájékoztathatja a hiba okáról.

2. módszer: a .bat futtatási kód módosítása

Az 1. módszer rendszeres használata nem túl praktikus. Ezért a két fent leírt parancs egyetlen állományban (melynek neve Windows-rendszereken ".bat-állomány") van megadva.

Amikor ezt az állományt futtatja, a benne lévő utasítások automatikusan végrehajtódnak. Ezért az indító parancs módosításához elegendő ezt az állományt módosítania. Az OmegaT ezen állományból való elindításakor sor kerül az új parancsok végrehajtására.

Parancsargumentumok futtatása

Az alapvető parancs leírását lásd fentebb. A parancs módosítása "argumentumok" hozzáadását jelenti. Az argumentumok helye a kezdő "java" és az azt követő "-jar OmegaT.jar" rész között van.

Alább megadjuk a lehetséges argumentumok felsorolását. További részleteket olvashat el az argumentumokkal kapcsolatosan, ha a terminálablakban beírja a man java parancsot. A man olyan parancs, amellyel a kiválasztott program kézikönyve jeleníthető meg. Előfordulhat, hogy rendszerén a java a saját nyelvén nem érhető el.

-Duser.language=XX

Amikor az OmegaT argumentumok nélkül indul el, a program először kideríti a felhasználó operációs rendszerének a nyelvét. Ha létezik ezen a nyelven felhasználói felület, akkor az OmegaT ezt használja. Ha tehát a felhasználó operációs rendszere orosz és az OmegaT le van fordítva oroszra, akkor az OmegaT orosz felhasználói felülettel, orosz nyelvű menükkel stb. jelenik meg. Ha felhasználó rendszerének megfelelő nyelvi változat nem áll rendelkezésre, akkor az OmegaT angol nyelven lesz használható. Ez az alapbeállítása.

A "-Duser.language=XX" argumentummal az OmegaT a megadott nyelvet fogja használni a felhasználó operációs rendszerének nyelve helyett. A parancsban az "XX" a használni kívánt nyelv kétbetűs kódját jelenti. Ha francia felülettel szeretné használni az OmegaT-t (például orosz operációs rendszeren), akkor a következő parancsot kell beírnia:

java -Duser.language=fr -jar OmegaT.jar

-Duser.country=XX

A nyelv mellett megadható az ország is, pl. kínai esetén CN vagy TW. Például a helyes Útmutató az azonnali kezdéshez megjelenítéséhez mind a nyelvet, mind az országot meg kell adnia. Erre akkor is szükség van, ha csak egy nyelvi kombináció áll rendelkezésre, mint pl. a pt_BR a portugál / brazil esetén. .

-Dswing.aatext=true

Ez az argumentum a betűkészleteken élfinomítást végez, így javítja megjelenésüket.

-XmxZZM

Ezzel a paranccsal több memóriát rendelhet az OmegaT-hez. Alaphelyzetben 256 Mb van hozzárendelve, ezért ennél alacsonyabb érték beállításának nincs értelme. A "ZZ" a hozzárendelt memóriát jelzi megabájtban. Így ha az OmegaT-t 1024 Mb (1 gigabájt) memóriával kívánja elindítani, akkor a parancs:

java -Xmx1024M -jar OmegaT.jar

Az argumentumokat kombinálhatja. Ha az OmegaT-t a fenti példának megfelelően kívánja futtatni, ezt a parancsot kell beírnia:

java -Dswing.aatext=true -Duser.language=pt -Duser.country=BR -Xmx1024M -jar OmegaT.jar

Az OmegaT forráskódból való előállítása

Melyik csomagot töltsem le?

Töltse le az OmegaT_1.7.3_Source.zip állományt
Ez a csomag tartalmazza az adott változat forrásállományait, és bármely rendszeren használható, ha azon telepítve van a Java 1.4 JDK vagy annak megfelelő Java-rendszer. Az OmegaT előállításához szüksége lesz az ant nevű segédprogramra is.
A legfrissebb forráskód elérhető a CVS-szerverről is:
cvs -z3 -d:pserver:anonymous@omegat.cvs.sourceforge.net:/cvsroot/omegat co -P omegat

Az OmegaT előállítása

Csomagolja ki az OmegaT_1.7.3_Source.zip állományt és adja meg az OmegaT_1.7.3_Source könyvtárat vagy írja be az ./omegat/ könyvtárat a CVS kódállománya számára. Ellenőrizze, hogy a könyvtárban van-e build.xml nevű állomány. Ezután a parancssorba írja be:

$ ant jar release

Ezzel létrejön a ./dist/ könyvtárban az OmegaT teljes változata, amelyben megtalálja az OmegaT futtatásához szükséges összes állományt.
