Systemanforderungen

OmegaT benötigt eine Java Runtime Environment (auch bekannt als JRE, J2RE oder ähnliche Bezeichnung). Wenn auf Ihrem System Java-Programme laufen, dann wird auch OmegaT darauf laufen. Versionen der Java Runtime Environment sind für die folgenden Plattformen erhältlich: Windows 95/98/2000/ME/XP, UNIX, Linux und Solaris (sowohl x86 als auch Sun SPARC). Bei Mac OS X ist die JRE bereits vorinstalliert.

Für diese Version von OmegaT wird die JRE-Version 1.4.2 oder höher empfohlen. Zwar wird das Programm wahrscheinlich mit allen 1.4er-Version funktionieren, es kann aber sein, dass manche Funktionen fehlen.

Nach dem aktuellen Wissensstand des OmegaT-Projekt-Teams funktioniert OmegaT NICHT auf den folgenden Plattformen:

  • Windows NT; die Java-Implementierung für NT ist fehlerhaft
  • Mac OS 9; die einzige hierfür erhältliche Java Runtime Environment (bekannt als MRJ) bietet nicht die benötigten Funktionen.
