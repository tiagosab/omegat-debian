Для пользователей Linux (Intel)

Какой пакет скачивать?

Установлена ли в вашей системе Java, совместимая с SUN's Java 1.4 JRE?

Да: 

скачивайте OmegaT_1.8.X_Without_JRE.zip. Этот пакет можно использовать на любой платформе, где установлена Java 1.4 JRE-совместимая JRE.

Нет / Я не знаю:

скачивайте OmegaT_1.8.X_Linux.tar.gz. В этот пакет включена SUN's Java Runtime Environment. Эта JRE не будет конфликтовать с другими инсталляциями Java, которые могут быть уже установлены на вашей системе.

Установка OmegaT

Чтобы установить OmegaT, просто распакуйте скачанный файл. В текущем каталоге будет создан каталог omegat/, в котором вы найдёте все файлы, необходимые для запуска OmegaT. Чтобы распаковать файл .tar.gz:

$ tar xf downloaded_file.tar.gz

Добавление OmegaT в ваши меню (KDE) или панели (Gnome)

Для пользователей KDE

Вы можете добавить OmegaT в меню следующим образом:

  • Центр Управления KDE - Рабочий стол - Панели - Меню - Редактировать меню К - Файл - Новый элемент/Новое подменю.

  • Затем, после выбора подходящего меню, добавьте новый элемент или подменю при помощи Файл - Новое подменю или Файл - Новый элемент. Введите OmegaT в качестве имени нового элемента.

  • В поле "Команда" найдите скрипт, запускающий OmegaT (файл с именем OmegaT в распакованном каталоге), и выберите его.

  • Щелкните на кнопке выбора иконки (справа от полей Имя/Описание/Комментарий) - Прочие значки - Обзор и зайдите в подкаталог /image в каталоге OmegaT. Выберите иконку OmegaT.png.

  • Сохраните изменения при помощи Файл - Сохранить.

Для пользователей Gnome

Вы можете добавить OmegaT в вашу панель (меню вверху экрана) следующим образом:

  • Щелкните правой кнопкой мыши на панели - Add New Launcher.

  • Введите "OmegaT" в поле "Name"; используйте кнопку обзора, чтобы ввести путь к запускающему OmegaT скрипту в поле "Command". Выберите скрипт и подтвердите выбор нажатием кнопки OK.

Запуск OmegaT

После установки OmegaT вы можете запускать программу напрямую из командной строки, создать скрипт с параметрами запуска или просто щёлкнуть мышкой на файле OmegaT.jar. Метод зависит от дистрибутива. Убедитесь, что у вас корректно настроена переменная PATH и что файлы .jar ассоциированы с исполняемым файлом Java.

Для пользователей Windows

Установлена ли в вашей системе Java, совместимая с SUN's Java 1.4 JRE?

Да:

скачивайте OmegaT_1.8.X_Windows_without_JRE.exe. Этот пакет можно использовать на любой Windows, где установлена Java 1.4 JRE-совместимая JRE.

скачивайте OmegaT_1.8.X_Windows.exe. В этот пакет включена SUN's Java Runtime Environment. Эта JRE не будет конфликтовать с другими инсталляциями Java, которые могут быть уже установлены на вашей системе.

Установка OmegaT

Чтобы установить OmegaT, дважды щёлкните на программе, которую вы скачали и следуйте инструкциям на экране.

Добавление OmegaT в стартовое меню или на рабочий стол

Во время установки программа спросит вас, хотите ли вы создать папку в меню "пуск" и ярлыки на рабочем столе и на панели быстрого запуска.

Вы можете создать ярлык и позднее, перетащив OmegaT.exe на рабочий стол или в меню "пуск".

Запуск OmegaT

После установки OmegaT вы можете щёлкнуть мышкой на файле OmegaT.jar, чтобы запустить программу напрямую, или же запускать её прямо из командной строки.

Однако, самый простой пусть - запустить программу OmegaT.exe В этом случае параметры будут прочитаны из файла OmegaT.l4J.ini, который находится в том же каталоге, что и exe-файл. Вы можете отредактировать его под свои настройки. Вот такой INI-файл зарезервирует для OmegaT 1 Gb памяти, и установит французский в качестве языка пользователя и Португалию как страну языка:

# OmegaT.exe runtime configuration
#
# To use a parameter, remove the '#' before the '-'
# Memory
-Xmx1024M
# Language
-Duser.language=FR
# Country
-Duser.country=PT
# Anti-aliasing (Java 1.4 and 1.5 only)
#-Dswing.aatext=true
# Entity expansion for XML (Java 1.4 only)
#-DentityExpansionLimit=400000

Для пользователей MacOSX

Какой пакет скачивать?

Скачивайте OmegaT_1.8.X_MacOSX.zip.

Установка OmegaT

Дважды щёлкните на файле OmegaT_1.8.X_MacOSX.zip, чтобы распаковать его. Образ диска OSX (OmegaT.dmg) смонтирует себя и покажет 2 файла: documentation.html и OmegaT.app. Скопируйте эти файлы в подходящий каталог (например, Applications, в котором вы можете создать подкаталог OmegaT). Сделав это, вы можете удалить файл OmegaT_1.8.X_MacOSX.zip.

Добавление OmegaT в Док

Добавьте OmegaT в Док, перетащив туда файл OmegaT.app.

Запуск OmegaT

Дважды щёлкните на OmegaT.app или на его ярлыке в Доке.

Если вы хотите изменить параметры запуска Java из командной строки, мы рекомендуем вам использовать пакет, доступный для "других систем", см. ниже. Кроме того, возможно напрямую изменять файл Info.plist, расположенный внутри пакета OmegaT.app.

Какой пакет скачивать?

Да:

скачивайте OmegaT_1.8.X_Without_JRE.zip Этот пакет можно использовать на любой платформе, где установлена Java 1.4 JRE-совместимая JRE.

откройте терминал и наберите "Java -version". Если появилось сообщение типа "command not found", то скорее всего, Java на вашей системе не установлена.

Следуйте инструкциям по установке необходимых пакетов.

Запуск OmegaT

После установки OmegaT вы можете запускать программу напрямую из командной строки, создать скрипт с параметрами запуска или просто щёлкнуть мышкой на файле OmegaT.jar. Метод зависит от дистрибутива. Убедитесь, что у вас корректно настроена переменная PATH и что файлы .jar ассоциированы с исполняемым файлом Java.

Запуск из командной строки

Обычно нет необходимости запускать OmegaT из командной строки. Тем не менее, такой метод запуска позволяет изменять некоторые аспекты работы программы. Есть два способа запустить OmegaT, используя командную строку.

Пользователям пакета Mac OSX, которые хотят управлять OmegaT из командной строки, мы рекомендуем использовать пакет для "Других систем". Смотри выше.

Метод 1: открытие окна командной строки

Окно командной строки также называется "окно терминала". Под Windows оно называется "окно MS-DOS" и доступно через меню "Пуск - Программы - MS-DOS". Его эквивалентом под Mac OSX является приложение Terminal, расположенное в Applications → Utilities.

Чтобы запустить OmegaT, обычно нужно отдать две команды. Первая:

cd {каталог}

где {каталог} это имя каталога (с полным путём к нему), где находится ваша OmegaT - точнее, файл OmegaT.jar. Следовательно, эта команда будет, скорее всего, выглядеть так:

Под Windows
cd C:\Program Files\OmegaT
Под Mac OSX
cd /Applications/OmegaT
В Linux
cd /usr/local/omegat

Эта команда изменяет текущий каталог на каталог, содержащий исполняемый файл OmegaT. Вторая команда собственно запускает OmegaT. В самом простом виде она такова:

java -jar OmegaT.jar

Обратите внимание на регистр букв - он имеет значение.

Этот метод удобен при отладке: если во время работы произошла ошибка, то сообщение о ней выведется в терминальное окно, что может дать вам полезную информацию.

Метод 2: изменение .bat-файла / стартового скрипта

Метод 1 довольно непрактичен для повседневного запуска программы. Поэтому две вышеупомянутые команды содержатся в файле ("скрипте", на Windows-системах также известен как ".bat-файл").

Когда запускается этот файл, то автоматически выполняются все команды внутри него. Следовательно, чтобы изменить команду запуска, достаточно изменить этот файл. Когда OmegaT будет запущена из этого файла, все новые команды тоже будут исполнены.

Аргументы команды запуска

Основные команды уже были перечислены выше. Изменения этих команд подразумевают добавление к ним "аргументов". Аргументы добавляются после слова "java" и до слов "-jar OmegaT.jar". Имейте в виду, что в Windows вы можете изменить файл OmegaT.l4J.ini под свои настройки.

Ниже приведён список возможных аргументов. Вы можете получить больше информации об аргументах, если отдадите в терминальном окне команду man java. man - это команда, которая отображает справку по выбранной команде. В вашей системе может и не быть справки для команды java на вашем языке.

Язык пользовательского интерфейса: -Duser.language=XX

     

Обычно, то есть, когда OmegaT запускается без каких-либо аргументов, программа сама определяет язык операционной системы пользователя. Если существует пользовательский интерфейс на этом языке, то OmegaT использует его. Таким образом, если операционная система пользователя настроена на русский и OmegaT была локализована на этот язык, то OmegaT будет работать с русским интерфейсом пользователя, русскими меню и т.д. Если определить язык системы пользователя не удалось, OmegaT запустится на английском. Это стандартное поведение.

Аргумент "-Duser.language=XX" указывает OmegaT использовать вместо языка операционной системы какой-то другой язык. "XX" в этой команде означает двухбуквенный код необходимого языка. Итак, например, чтобы запустить OmegaT с французским интерфейсом на русской операционной системе, нам понадобится следующая команда: java -Duser.language=fr -jar OmegaT.jar

Настройка страны: -Duser.country=XX

     

Кроме языка вы можете указать страну, например, CN или TW в случае с китайским языком. Чтобы получить корректное Руководство по Быстрому Старту, вам нужно указать и язык, и страну. Это необходимо, даже если доступна лишь одна комбинация, например, pt_BR в случае с португальским / бразильским.

Сглаживание шрифтов: -Dswing.aatext=true

     

Этот аргумент включает сглаживание (anti-aliasing) шрифтов, тем самым улучшая их внешний вид.

Выделение памяти: -XmxZZM

     

Эта команда выделяет OmegaT больше памяти. По умолчанию назначается 256 MB, так что назначать меньше нет никакого смысла. "ZZ" обозначает объём назначаемой памяти в мегабайтах. Таким образом, чтобы запустить OmegaT с выделением 1024 MB (1 гигабайт) памяти, нужно отдать следующую команду: java -Xmx1024M -jar OmegaT.jar

Аргументы можно комбинировать: чтобы запустить OmegaT со всеми вышеперечисленными примерами, нужно отдать такую команду:
java -Dswing.aatext=true -Duser.language=pt -Duser.country=BR -Xmx1024M -jar OmegaT.jar

Компилирование OmegaT из исходников

Какой пакет скачивать?

Скачивайте OmegaT_1.8.X_Source.zip
Этот пакет содержит файлы с исходными кодами данной версии программы. Его можно использовать на любой системе с Java 1.4 JDK или совместимой Java-системой. Кроме того, для сборки OmegaT вам понадобится утилита ant.
Для контроля версий OmegaT использует Subversion. Код программы с последними обновления доступен на SVN-сервере:
https://omegat.svn.sourceforge.net/svnroot/omegat/trunk

Сборка OmegaT

Распакуйте файл OmegaT_1.8.X_Source.zip и войдите в каталог OmegaT_1.8.X_Source или в каталог ./omegat/, если вы скачали код из SVN. Проверьте, что в этом каталоге есть файл build.xml. Затем в командной строке наберите:

$ ant jar release

Эта команда создаст полный пакет OmegaT в каталоге ./dist/, где вы найдёте все файлы, необходимые для запуска OmegaT.
